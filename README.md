# robot-project

## 项目简介
本项目基于 MATLAB 实现：
- 7自由度机械臂运动学建模
- 5自由度欠驱动手手势控制
- 轨迹规划
- 人机交互（石头剪刀布）/对一些基本手势做出回应

## 项目结构
- arm：机械臂建模
- hand：欠驱动手控制
- trajectory：轨迹规划
- vision：视觉识别
- demo：最终演示

## 开发环境
- MATLAB
- Robotics Toolbox
- Image Processing Toolbox

## 草稿
手势：比心、数字、打招呼
